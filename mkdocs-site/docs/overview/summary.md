# Executive Summary

## Project Background

Learning Through Motion needed a complete website rebuild to:

1. **Modernize** their outdated website
2. **Showcase** their three core SEND programmes effectively
3. **Improve accessibility** for their target audience (schools, SENCOs, parents)
4. **Drive enquiries** from schools across Greater Manchester and Lancashire
5. **Add interactive elements** for children with SEND

## What We Delivered

### A Complete Next.js Website

Built from the ground up with modern web technologies:

- **13 unique pages** covering all aspects of the business
- **20+ custom components** for consistent UI/UX
- **4 interactive games** designed for SEND learners
- **Fully responsive** design (mobile, tablet, desktop)
- **WCAG AAA accessibility** compliance throughout

### Key Achievements

!!! success "Accessibility Excellence"
    - **15.8:1 contrast ratio** (WCAG AAA requires 7:1)
    - Atkinson Hyperlegible font for dyslexia support
    - Full keyboard navigation
    - Skip navigation links
    - Semantic HTML throughout

!!! success "Modern Architecture"
    - Next.js 16 with React 19
    - TypeScript strict mode
    - Server Components for performance
    - Centralized content management
    - No database dependencies (static content)

!!! success "User Experience"
    - Dynamic image carousels
    - Interactive testimonials with modals
    - Multiple clear CTAs
    - Sticky enquiry button
    - Programme-specific accent colors
