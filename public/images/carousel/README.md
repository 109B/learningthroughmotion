# Carousel Images

This directory contains the image folders for the carousels on each sub-page.

## How to add images

1.  **Navigate** to the folder for the page you want to update (e.g., `maths`, `sensory`, `coaches`).
2.  **Add your images** to the folder.
3.  **Rename** your images to match the placeholder names in the code, OR update the code to match your new filenames.
